About This Model

AI-powered medical imaging analysis for bone fracture detection

üéØ Project Overview

This X-ray Bone Fracture Detection System is an AI-powered web application designed to assist medical professionals in identifying bone fractures from X-ray images. Using state-of-the-art deep learning technology, the system provides rapid, accurate analysis with visual explanations.

Technology Stack

  • Deep Learning: TensorFlow & Keras
  • Model: DenseNet121 (Transfer Learning)
  • Image Processing: OpenCV
  • Visualization: Grad-CAM
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript

Performance Metrics

  • Accuracy: 92-95%
  • Precision: 88-90%
  • Recall: 94-96%
  • Training Data: 40,000+ X-rays
  • Dataset: MURA (Stanford)
  • Inference Time: < 2 seconds

Target Users

  • Emergency room doctors
  • Radiologists
  • Orthopedic surgeons
  • Rural healthcare facilities
  • Medical students (education)
  • Telehealth providers

üî¨ How It Works

  1. Image Upload: User uploads an X-ray image through the web interface
  2. Preprocessing: Image is resized, denoised, and normalized using OpenCV
  3. AI Analysis: DenseNet121 CNN processes the image and predicts fracture probability
  4. Grad-CAM Visualization: If fracture detected, system highlights the specific location
  5. Results Display: Prediction, confidence score, and visualization shown to user

üìö Training Data

The model is trained on the MURA (Musculoskeletal Radiographs) dataset from Stanford University, containing over 40,000 musculoskeletal radiographs. The dataset includes images of 7 body parts: elbow, finger, forearm, hand, humerus, shoulder, and wrist.

Reference: Rajpurkar, P. et al. "MURA: Large Dataset for Abnormality Detection in Musculoskeletal Radiographs." arXiv preprint arXiv:1712.06957 (2017).

üé® Grad-CAM Explanation

Grad-CAM (Gradient-weighted Class Activation Mapping) is a visualization technique that shows which regions of an image are most important for the AI's prediction. In our system, it highlights the areas where the model detected potential fractures, making the AI's decision-making process transparent and trustworthy.

Red/yellow areas indicate high confidence regions where fractures are likely present, while blue/green areas show regions of lower importance to the prediction.
